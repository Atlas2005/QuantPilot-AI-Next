# Success Metrics

QuantPilot-AI 2.0 should be measured by evidence quality and improved probability of real profitability, not by early implementation volume.

## Metrics

- trustworthy data
- reproducible research
- realistic A-share market rules
- stable contracts
- replaceable adapters
- OOS evidence
- walk-forward evidence
- paper feedback
- risk-adjusted performance
- commercial/license safety
- reduced overfitting
- improved probability of real profitability

## Non-Metrics

- number of custom modules
- number of dependencies installed
- backtest performance without realism checks
- agent complexity before foundations are stable
- live-trading capability before readiness review
