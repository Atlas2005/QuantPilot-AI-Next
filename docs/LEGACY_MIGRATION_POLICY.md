# Legacy Migration Policy

The old QuantPilot-AI v2 project is a legacy research archive only.

## Allowed Legacy Uses

- lessons learned
- safety and guardrail reference
- CI/reference discipline
- A-share constraint history
- failure history
- architecture-debt history

## Prohibited Legacy Uses

- copying old v2 source code as new core
- migrating the old custom backtester as new core
- migrating the old ML trainer as new core
- migrating the old factor builder as new core
- migrating the old order generator as new core
- migrating custom execution modules as new core

Any future legacy reference must be explicitly reviewed and scoped.
