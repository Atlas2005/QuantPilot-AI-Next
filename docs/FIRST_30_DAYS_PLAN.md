# First 30 Days Plan

## Goal

Build planning discipline, candidate evaluation structure, initial contracts, and safe local validation before any high-risk implementation.

## Suggested Sequence

1. Complete Phase 0A planning package.
2. ChatGPT reviews Phase 0A documents.
3. Complete Phase 0B clean skeleton and minimal CI.
4. Create Phase 1 open-source candidate registry.
5. Begin Phase 2 core contract planning.
6. Defer agent orchestration, data fetching, market data integrations, broker paths, and trading logic.

## Guardrails

The first 30 days should optimize for correctness, governance, and replaceability rather than speed of trading feature creation.
