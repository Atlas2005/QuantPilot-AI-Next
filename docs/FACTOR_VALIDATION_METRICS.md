# Factor Validation Metrics

Phase 7B creates a minimal validation metrics foundation for factor research. It is a shape and policy layer only.

This phase cannot claim alpha, profitability, or statistical significance.

## Toy IC-Like Metric

The toy information coefficient-like metric is a rank correlation between toy factor values and toy forward returns.

It is computed only on the fake local fixture. It does not prove predictive power, robustness, capacity, or profitability.

## Grouped Forward Returns

Grouped forward returns sort toy factor values and split them into simple quantile-like groups. The output is only a shape check for future factor reports.

Tiny fake fixtures cannot prove anything about real performance.

## Evidence Required Later

Before any real alpha claim, the project needs:

- larger real historical dataset
- multiple symbols
- OOS split
- walk-forward validation
- transaction costs
- A-share execution constraints
- turnover and capacity checks
- paper feedback
- risk-adjusted review
- repeated evidence across market regimes

## Deferred External Analytics

Phase 7B does not integrate:

- Alphalens Reloaded
- quantstats
- empyrical / empyrical-reloaded
- Qlib

These remain future candidates after contracts, local metrics, data quality, and validation requirements are stable.

## Phase 7C Candidate Library Relationship

Phase 7B metrics are evidence gates for Phase 7C candidates.

Adding more factor candidates does not bypass validation requirements. Every candidate remains toy or not evaluated until it passes larger-data checks, OOS validation, walk-forward validation, transaction cost review, A-share execution constraints, capacity checks, and paper feedback.

## Phase 7D External Analytics Preflight

External analytics remain deferred.

Alphalens-like tools require real factor and forward-return panels. quantstats and empyrical require meaningful return streams. No external analytics output exists yet.

## Phase 7E Real Data Readiness

Real-data readiness is required before moving from toy metrics to real metrics.

OOS, walk-forward validation, transaction costs, A-share execution rules, and reproducibility are mandatory before real alpha claims.

## Current Limitations

- fake fixture only
- tiny sample
- no transaction costs
- no A-share execution rules
- no OOS
- no walk-forward
- no paper feedback
- no statistical significance
- no profitability evidence
