# Open-Source Candidate Universe

This file records candidate projects only. Step 0A does not install, import, integrate, or select them.

## Classification System

Each candidate should later be classified as one of:

- `adopt_directly`
- `wrap_with_adapter`
- `borrow_architecture_only`
- `prototype_required`
- `defer_until_foundation_ready`
- `avoid_for_now`
