# Project Positioning

QuantPilot-AI 2.0 is an A-share AI quant integration platform.

It is open-source-first, adapter-first, contract-first, A-share-first, Python-first, and polyglot-ready.

The project is not a continuation of the old v2 custom Codex-generated architecture. The old project is a legacy research archive only.

## Product Goal

The long-term goal is to build a money-making AI quantitative trading platform. That goal is profitability-oriented but evidence-gated.

No component may claim profitability or trading readiness without clean data, realistic market rules, sample-out validation, walk-forward/OOS evidence, paper feedback, and risk-adjusted performance review.

## Unique Value

The project should create value through A-share adaptation:

- T+1
- 100-share lot size
- limit-up/limit-down
- suspension handling
- ST treatment
- fees
- stamp duty
- slippage
- liquidity constraints
- small-capital constraints
- data quality problems
- realistic backtest and paper feedback loop

## Current Boundary

Phase 0A creates planning documents only. It does not create implementation code.
